== Description ==

Shortcode to display real time train information from the National Rail API.


== Installation ==

1. Upload `tq-nationalrail-api-client.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place a `[tq_nationalrail]` shortcode in your page

== Shortcode parameters ==

1. station - Three-letter CRS code of the station. List here:  https://www.nationalrail.co.uk/stations_destinations/48541.aspx
(All other parameters are hard-coded at the moment)
